# symfonycast course downloader!

Un scrapercito para descargar un curso desde [Symfonycast](https://symfonycast.com).

## instrucciones
* clona el repo
* crea un .env con la siguiente estructura
```env
USERNAME=<correo>
PASSWORD=<contraseña>
```
* `npm install`
* `npm run start course <course-url>`
